# Mayday for Windows 1.0.0

## What's New

- First public Windows desktop release of Mayday.
- Added a focused Flutter interface for importing access data, checking readiness, and controlling the protected connection.
- Added system tray support with connected and disconnected status icons.
- Added close-to-tray behavior and single-window activation when Mayday is launched again.
- Added English and Russian interface language support.
- Added optional app autostart after Windows sign-in.
- Added Windows app selection controls for connection preferences.
- Added a diagnostics area with user-facing readiness messages.
- Added a guided administrator restart flow when elevated permissions are required.
- Added a Windows installer package for release distribution.

## Fixes

- Improved connection startup and shutdown feedback.
- Improved behavior when exiting from the tray so the active connection is stopped before the app closes.
- Improved local data protection for imported access data.
- Improved import handling for current and legacy access data.
- Improved diagnostics messages so user-facing errors are clearer.
- Improved tray icon quality and size for better visibility on Windows.
- Improved Windows integration for smoother everyday app use.

## Compatibility

- Supports Windows 10 and Windows 11 on 64-bit PCs.
- Requires the permissions requested by the app for connection control.
- Autostart opens Mayday after sign-in, but does not start a connection automatically.

# Mayday для Windows 1.0.0

## Что нового

- Первый публичный desktop-релиз Mayday для Windows.
- Добавлен понятный Flutter-интерфейс для импорта данных доступа, проверки готовности и управления защищенным подключением.
- Добавлена поддержка системного трея с отдельными иконками для включенного и выключенного состояния.
- Добавлено закрытие окна в трей и открытие уже запущенного окна при повторном запуске Mayday.
- Добавлена поддержка русского и английского языка интерфейса.
- Добавлен опциональный автозапуск приложения после входа в Windows.
- Добавлены элементы выбора Windows-приложений для параметров подключения.
- Добавлен раздел диагностики с пользовательскими сообщениями о готовности.
- Добавлен сценарий перезапуска с правами администратора, когда приложению нужны повышенные права.
- Добавлен Windows-установщик для публикации релиза.

## Исправления

- Улучшена обратная связь при запуске и остановке подключения.
- Улучшен выход через трей: активное подключение останавливается до закрытия приложения.
- Улучшена защита локально сохраненных данных доступа.
- Улучшена обработка импорта актуальных и устаревших данных доступа.
- Улучшены сообщения диагностики, чтобы пользовательские ошибки были понятнее.
- Улучшены размер и качество иконок в трее для лучшей видимости в Windows.
- Улучшена Windows-интеграция для более плавного повседневного использования приложения.

## Совместимость

- Поддерживаются Windows 10 и Windows 11 на 64-битных ПК.
- Для управления подключением нужны права, которые запрашивает приложение.
- Автозапуск открывает Mayday после входа в Windows, но не включает подключение автоматически.
